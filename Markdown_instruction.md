## Списки

Чтобы Добавить ненумерованные списки, нуобходимо пункты выделить звездочкой(*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы Добавить нумерованные списки, нуобходимо пункты просто пронумеровать. Например, вот так:
1. Первый пункт
2. Второй пункт
